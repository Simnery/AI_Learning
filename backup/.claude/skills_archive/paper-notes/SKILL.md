# 论文阅读笔记

## 笔记模板

- **元信息**: 标题、作者、机构、年份、会议/期刊、arXiv 链接
- **一句话总结**: 这篇论文做了什么
- **核心贡献**: 3-5 个关键创新点
- **方法**: 核心方法/架构简述
- **实验**: 主要实验结果、对比基线
- **关键洞察**: 值得记住的设计选择或 trick
- **复现要点**: 复现时需要注意的细节
- **与我相关**: 对当前学习/项目的启发

## 规则

- 笔记保存为 `papers/<论文简称>.md`
- 用中文与用户交互
