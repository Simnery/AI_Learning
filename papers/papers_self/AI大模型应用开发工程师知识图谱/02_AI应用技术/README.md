# 2. AI应用技术

> **本章定位**：应用工程师核心技能层——RAG、Agent、微调三条主线，每节含可运行代码，是全图谱中**内容最厚**的一章。

## 学习目标

学完本章你应能：

- 从零搭建 RAG 流水线（解析→分块→Embedding→检索→生成）
- 实现 Function Calling / MCP / ReAct 类 Agent
- 完成 LoRA 微调或蒸馏的数据准备与训练流程
- 根据场景判断「RAG vs 长上下文 vs Agent 搜索」

## 建议学习顺序（约 4–5 周）

### 路线 A：应用开发（推荐）

| 阶段 | 节点 | 周次 |
|------|------|:---:|
| RAG 基础 | 2-1-1 → 2-1-2 → 2-1-4 | 1–2 |
| RAG 进阶 | 2-1-3 多模态 → 2-1-5 替代方案 | 2 |
| Agent | 2-2-1 → 2-2-2 → 2-2-3 → 2-2-4 | 3–4 |
| 微调 | 2-3-1 → 2-3-2 → 2-3-3（2-3-4 视觉可选） | 4–5 |

### 路线 B：先 Agent 后 RAG

适合已有检索基础的学员：2-2-1 起，再回补 2-1 系列。
