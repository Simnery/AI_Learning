## 核心概念

- **是什么**：图像识别是让计算机"看懂"图片——判断图里有什么（分类）、在哪里（检测）、每个像素属于什么（分割）。缺陷检测是工业场景的图像识别应用——自动找出产品表面的划痕、裂纹、污点等。
- **为什么重要**：图像识别是计算机视觉（CV）的基础能力，也是深度学习最早大规模落地的领域。工业缺陷检测每年能帮制造业节省数十亿质检成本。
- **前置知识**：PyTorch 基础（参考 [3-5-1 PyTorch核心概念](3-5-1_PyTorch核心概念.md)）、CNN 基本概念（卷积、池化）。

### 图像识别三大任务

```
分类: [猫]
┌────────────────┐
│  整张图 → 一个标签  │
└────────────────┘

检测: [猫 (x,y,w,h)]
┌────────────────┐
│  图上画框 + 标签  │
└────────────────┘

分割: 逐像素分类
┌────────────────┐
│ 每个像素一个类别  │
└────────────────┘
```

## 原理讲解

### 1. CNN 核心四层

| 层 | 作用 | 大白话 |
|----|------|--------|
| 卷积层 (Conv) | 用一个小窗口（3×3/5×5）在图上滑动提取特征 | "看局部纹理" |
| 批归一化 (BN) | 把每层输出调到标准分布，稳定训练 | "音量统一" |
| 激活层 (ReLU) | 引入非线性，负值清零 | "只保留正信号" |
| 池化层 (Pool) | 缩小特征图，降低计算量 | "浓缩精华" |

### 2. 缺陷检测三种方案

| 方案 | 原理 | 优点 | 缺点 |
|------|------|------|------|
| **分类法** | 将整图分为"良品/缺陷" | 简单快速 | 不知道缺陷在哪 |
| **检测法** | 用目标检测框出缺陷位置 | 精确定位 | 需要标注框 |
| **分割法** | 逐像素标记缺陷区域 | 最精细 | 标注成本最高 |

### 3. 卷积网络可视化

理解 CNN"看到了什么"的常用方法：

- **特征图可视化**：直接把中间层的输出画出来看
- **Grad-CAM**：用梯度生成热力图，叠加到原图上看模型关注哪里
- **滤波器可视化**：把第一层卷积核画出来（通常能看到边缘/颜色检测器）

### 示例1：CNN 图像分类器（CIFAR-10）

```python
import torch
import torch.nn as nn
import torch.optim as optim
import torchvision
import torchvision.transforms as transforms
from torch.utils.data import DataLoader

# 数据增强 + 归一化
transform_train = transforms.Compose([
    transforms.RandomHorizontalFlip(),    # 随机水平翻转
    transforms.RandomCrop(32, padding=4), # 随机裁剪
    transforms.ToTensor(),
    transforms.Normalize((0.4914, 0.4822, 0.4465), (0.247, 0.243, 0.261))
])
transform_test = transforms.Compose([
    transforms.ToTensor(),
    transforms.Normalize((0.4914, 0.4822, 0.4465), (0.247, 0.243, 0.261))
])

# 加载 CIFAR-10
trainset = torchvision.datasets.CIFAR10(root='./data', train=True, download=True, transform=transform_train)
testset = torchvision.datasets.CIFAR10(root='./data', train=False, download=True, transform=transform_test)
train_loader = DataLoader(trainset, batch_size=64, shuffle=True)
test_loader = DataLoader(testset, batch_size=64, shuffle=False)

classes = ('plane', 'car', 'bird', 'cat', 'deer', 'dog', 'frog', 'horse', 'ship', 'truck')

# CNN 模型
class SimpleCNN(nn.Module):
    def __init__(self, num_classes=10):
        super().__init__()
        self.features = nn.Sequential(
            # Block 1: 32×32 → 16×16
            nn.Conv2d(3, 32, 3, padding=1), nn.BatchNorm2d(32), nn.ReLU(),
            nn.Conv2d(32, 32, 3, padding=1), nn.BatchNorm2d(32), nn.ReLU(),
            nn.MaxPool2d(2), nn.Dropout(0.25),
            # Block 2: 16×16 → 8×8
            nn.Conv2d(32, 64, 3, padding=1), nn.BatchNorm2d(64), nn.ReLU(),
            nn.Conv2d(64, 64, 3, padding=1), nn.BatchNorm2d(64), nn.ReLU(),
            nn.MaxPool2d(2), nn.Dropout(0.25),
            # Block 3: 8×8 → 4×4
            nn.Conv2d(64, 128, 3, padding=1), nn.BatchNorm2d(128), nn.ReLU(),
            nn.MaxPool2d(2), nn.Dropout(0.25),
        )
        self.classifier = nn.Sequential(
            nn.Flatten(),
            nn.Linear(128 * 4 * 4, 256), nn.ReLU(), nn.Dropout(0.5),
            nn.Linear(256, num_classes)
        )
    
    def forward(self, x):
        x = self.features(x)
        return self.classifier(x)

device = torch.device("cuda" if torch.cuda.is_available() else "cpu")
model = SimpleCNN().to(device)
criterion = nn.CrossEntropyLoss()
optimizer = optim.AdamW(model.parameters(), lr=0.001)

# 训练
for epoch in range(10):
    model.train()
    running_loss, correct, total = 0.0, 0, 0
    for inputs, labels in train_loader:
        inputs, labels = inputs.to(device), labels.to(device)
        optimizer.zero_grad()
        outputs = model(inputs)
        loss = criterion(outputs, labels)
        loss.backward()
        optimizer.step()
        running_loss += loss.item()
        _, predicted = outputs.max(1)
        correct += predicted.eq(labels).sum().item()
        total += labels.size(0)
    print(f"Epoch {epoch+1:2d} | Loss: {running_loss/len(train_loader):.3f} | Acc: {correct/total:.3f}")

# 测试
model.eval()
correct, total = 0, 0
with torch.no_grad():
    for inputs, labels in test_loader:
        outputs = model(inputs.to(device))
        correct += outputs.argmax(1).eq(labels.to(device)).sum().item()
        total += labels.size(0)
print(f"\n测试准确率: {correct/total:.4f}")
```

### 示例2：Grad-CAM 可视化模型注意力

```python
import cv2
import numpy as np
import torch.nn.functional as F

class GradCAM:
    def __init__(self, model, target_layer):
        self.model = model
        self.target_layer = target_layer
        self.gradients = None
        self.activations = None
        
        # 注册 hook 捕获中间层的激活和梯度
        target_layer.register_forward_hook(self._save_activation)
        target_layer.register_full_backward_hook(self._save_gradient)
    
    def _save_activation(self, module, input, output):
        self.activations = output.detach()
    
    def _save_gradient(self, module, grad_input, grad_output):
        self.gradients = grad_output[0].detach()
    
    def generate(self, input_tensor, target_class=None):
        # 前向
        output = self.model(input_tensor)
        if target_class is None:
            target_class = output.argmax(1).item()
        
        # 反向——只对目标类计算梯度
        self.model.zero_grad()
        one_hot = torch.zeros_like(output)
        one_hot[0][target_class] = 1
        output.backward(gradient=one_hot, retain_graph=True)
        
        # 计算权重 = 梯度全局平均池化
        weights = self.gradients.mean(dim=(2, 3), keepdim=True)
        
        # 热力图 = Σ(权重 × 激活图)
        cam = (weights * self.activations).sum(dim=1, keepdim=True)
        cam = F.relu(cam)  # 只保留正贡献
        
        # 归一化到 [0, 1]
        cam = cam - cam.min()
        cam = cam / (cam.max() + 1e-8)
        
        return cam.squeeze().cpu().numpy()
```

### 示例3：工业缺陷检测——基于分类的简化方案

```python
import torch
import torch.nn as nn
from torch.utils.data import Dataset, DataLoader
from torchvision import transforms

# 模拟缺陷检测数据集
class DefectDataset(Dataset):
    def __init__(self, num_samples=1000, is_train=True):
        self.num_samples = num_samples
        self.is_train = is_train
    
    def __len__(self):
        return self.num_samples
    
    def __getitem__(self, idx):
        # 模拟 64×64 灰度图
        img = torch.randn(1, 64, 64)
        # 良品：均值接近 0；缺陷：均值偏离大
        is_defect = img.mean().abs() > 0.15
        label = 1 if is_defect else 0  # 1=缺陷, 0=良品
        return img, label

# 数据增强（对工业检测尤其重要——缺陷可能在任意位置/角度）
train_transform = transforms.Compose([
    transforms.RandomRotation(15),
    transforms.RandomHorizontalFlip(),
])

# 模型
class DefectCNN(nn.Module):
    def __init__(self):
        super().__init__()
        self.net = nn.Sequential(
            nn.Conv2d(1, 16, 3, padding=1), nn.BatchNorm2d(16), nn.ReLU(), nn.MaxPool2d(2),
            nn.Conv2d(16, 32, 3, padding=1), nn.BatchNorm2d(32), nn.ReLU(), nn.MaxPool2d(2),
            nn.Conv2d(32, 64, 3, padding=1), nn.BatchNorm2d(64), nn.ReLU(), nn.MaxPool2d(2),
            nn.AdaptiveAvgPool2d(1),  # 全局平均池化 → 1×1×64
            nn.Flatten(),
            nn.Linear(64, 32), nn.ReLU(), nn.Dropout(0.3),
            nn.Linear(32, 2)
        )
    
    def forward(self, x):
        return self.net(x)

# 训练
device = torch.device("cuda" if torch.cuda.is_available() else "cpu")
model = DefectCNN().to(device)
optimizer = torch.optim.AdamW(model.parameters(), lr=0.001)
criterion = nn.CrossEntropyLoss()

train_loader = DataLoader(DefectDataset(2000), batch_size=32, shuffle=True)
test_loader = DataLoader(DefectDataset(400), batch_size=32)

for epoch in range(5):
    model.train()
    for imgs, labels in train_loader:
        imgs, labels = imgs.to(device), labels.to(device)
        optimizer.zero_grad()
        loss = criterion(model(imgs), labels)
        loss.backward()
        optimizer.step()

# 评估
model.eval()
correct, total = 0, 0
with torch.no_grad():
    for imgs, labels in test_loader:
        preds = model(imgs.to(device)).argmax(1)
        correct += preds.eq(labels.to(device)).sum().item()
        total += labels.size(0)
print(f"缺陷检测准确率: {correct/total:.4f}")
```

## 进阶方向

### 1. 当前局限性

- CNN 感受野有限（只能看局部），远距离依赖需要很深网络 → Transformer 类模型（ViT/Swin）在大图上优势明显
- 缺陷检测的标注成本极高——半监督/自监督学习是降本关键
- 分类法只能判断有无缺陷，不能定位。工业场景通常需要目标检测或语义分割

### 2. 下一步学习

| 主题 | 说明 | 相关节点 |
|------|------|----------|
| YOLO 目标检测 | 实时检测，工业界最常用 | [3-5-4 YOLO](3-5-4_视觉检测模型YOLO.md) |
| U-Net 语义分割 | 像素级缺陷标注 | 医学图像/缺陷分割 |
| Vision Transformer | Transformer 在 CV 中的应用 | 论文阅读 |
| 自监督学习 | 减少标注依赖 | SimCLR/MAE |

### 3. 工业界最佳实践

- **数据为中心**：工业场景下，数据质量和覆盖率比模型 fancy 程度重要 10 倍
- **数据增强必做**：旋转、翻转、亮度/对比度、噪声——工业拍摄环境变化大
- **正负样本均衡**：缺陷通常是少数（<5%），需要用 Focal Loss 或过采样

## 常见问题

### 小白最常踩的坑

1. **图片尺寸不一致**：CNN 要求固定输入尺寸。用 `transforms.Resize((H, W))` 统一，或用 `AdaptiveAvgPool2d` 做全局池化
2. **通道顺序搞反**：PyTorch 用 (B, C, H, W)，OpenCV 用 (H, W, C)，转换时注意 `torch.permute` 或 `np.transpose`
3. **归一化不对**：ImageNet 预训练模型要求特定均值和标准差。不用预训练权重的话，0-1 归一化就行

### 自检题

**Q1**：CNN 的四个核心层（卷积、BN、ReLU、池化）各自起什么作用？

<details><summary>答案</summary>

- 卷积：提取局部特征（边缘、纹理等）
- BN（批归一化）：稳定各层输入分布，加速收敛，允许更大学习率
- ReLU：引入非线性，缓解梯度消失
- 池化：下采样，降低计算量，增加感受野
</details>

**Q2**：工业缺陷检测中"分类法"和"检测法"的区别是什么？分别适合什么场景？

<details><summary>答案</summary>

- 分类法：整图输出"良品/缺陷"标签。适合：缺陷类型不重要、只需判断是否合格的场景
- 检测法：用框标出缺陷位置和类型。适合：需要知道缺陷在哪、什么类型的场景
</details>

**Q3**：Grad-CAM 是如何生成热力图的？核心思想是什么？

<details><summary>答案</summary>

核心思想：用目标类的梯度（∂score/∂A）衡量每个特征图通道对最终决策的"贡献度"，然后加权求和所有通道的激活图得到热力图。热力图高亮区域就是模型做决策时最关注的图像区域。
</details>
