# 1. AI大模型基础

> **本章定位**：零基础入门层——建立「大模型是什么、怎么用、怎么部署」的全局认知，为后续 RAG/Agent/项目实战打底。

## 学习目标

学完本章你应能：

- 用大白话解释 Prompt、RAG、Agent 的区别与适用场景
- 独立调用至少一种大模型 API（OpenAI 兼容或 DashScope）
- 说出企业部署时 Ollama / vLLM / SGLang 的选型依据
- 理解 KV Cache、PagedAttention 与高并发推理的关系
