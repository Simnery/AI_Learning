# 6. 专项面试辅导课

> **本章定位**：求职冲刺层——在 1–5 章学完后，把知识转化为**简历 bullet + 面试话术 + 模拟追问**。

## 学习目标

学完本章你应能：

- 用 STAR 法则写出 2–3 段可量化的项目经历（RAG / Agent / 框架 / 微调各 1 段）
- 回答 10+ 道高频面试题并承受 2 层追问
- 使用文档中的「面试模拟器」代码做自主练习

| 顺序 | 节点 | 对应正文章节 |
|:---:|------|-------------|

每篇流程：**读面试题要点 → 写自己的 STAR 答案 → 跑模拟器代码 → 录音回放 15 分钟**。

## 先修

- **必须完成**：[第 2 章](../02_AI应用技术/README.md) 对应模块 + 至少 1 个 [第 5 章](../05_项目实战/README.md) 项目

## 简历模板（STAR 骨架）

```
【项目】企业知识库 RAG（第 5 章 5-1）
S：某企业客服文档 10 万页，人工检索平均 15 分钟
T：搭建可溯源的 RAG，目标 Top-3 命中率 >85%
A：Docling 解析 + M3E Embedding + LanceDB + 混合检索 + Rerank
R：命中率 88%，P95 延迟 180ms，人工介入率下降 40%
```

将 `{}` 内换成你的真实数据；无数据时用「沙箱实验 + 指标定义」诚实表述。
