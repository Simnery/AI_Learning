# 5. 项目实战

> **本章定位**：综合落地层——把 2–4 章技术串进真实业务（RAG 大赛、Hermes Agent、换脸、Wiki、昇腾部署）。**建议第 2、3 章完成后再学。**

## 学习目标

学完本章你应能：

- 复述企业 RAG 冠军项目的架构与调优要点
- 描述 Hermes 长期记忆（4 层）+ 多 Agent 协作的端到端数据流
- 理解 GPU 队列调度、换脸流水线等工程问题
- （可选）在昇腾 NPU 上完成 DeepSeek 部署与本地 Agent 集成
